# 🤖 AI Temperature Comparison Workflow

An **n8n automation workflow** that compares how different **AI temperature settings** influence the responses generated by the same prompt.

The workflow executes the prompt across multiple AI agents configured with **Temperature 0.0, 0.3, 0.7, and 1.0**, then consolidates the outputs into **Google Sheets** for side-by-side comparison and analysis.

---

## 🚀 Features

- Compare AI responses across multiple temperature settings
- Parallel execution using n8n AI Agents
- Automatic response merging
- Google Sheets integration for result storage
- Easy side-by-side output analysis

---

## 🛠️ Tech Stack

- **n8n**
- **Google Gemini**
- **Google Sheets**
- **AI Agents**

---

## 🔄 Workflow

```
Prompt
   │
   ▼
Split Input
   │
   ├── AI Agent (Temp 0.0)
   ├── AI Agent (Temp 0.3)
   ├── AI Agent (Temp 0.7)
   └── AI Agent (Temp 1.0)
          │
          ▼
      Merge Results
          │
          ▼
    Google Sheets
```

---

## 🌡️ Temperature Overview

| Temperature | Behavior | Best For |
|-------------|----------|----------|
| **0.0** | Deterministic and consistent | Coding, factual tasks |
| **0.3** | Slightly creative | Summaries, emails |
| **0.7** | Balanced | Content creation, brainstorming |
| **1.0** | Highly creative | Storytelling, ideation |
