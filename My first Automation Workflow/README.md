
# 📧 Feedback Email Automation using n8n

An event-driven workflow built with **n8n** that automatically sends a thank-you email whenever a user submits a feedback form.

This project helped me explore workflow automation, self-hosting, OAuth authentication, and event-driven processes using n8n.

---

## 🚀 Project Overview

The workflow listens for feedback submitted through an n8n Form Trigger.

Once a user submits the form, the workflow automatically sends a thank-you email to the email address provided by the user.

This eliminates the need to manually acknowledge every feedback submission.

## 📌 Workflow

```text
User submits Feedback Form
            │
            ▼
     n8n Form Trigger
            │
            ▼
Extract User Email Address
            │
            ▼
 Gmail Node sends
Thank-you Email Automatically
```

## ⚙️ Setup Guide

### 1. Clone Repository

```bash
git clone https://github.com/yourusername/Feedback-Email-Automation.git
```

---

### 2. Install Docker

Download Docker Desktop and make sure it is running.

---

### 3. Start n8n

```bash
docker run -it --rm \
-p 5678:5678 \
-v ~/.n8n:/home/node/.n8n \
docker.n8n.io/n8nio/n8n
```

---

### 4. Import Workflow

Open n8n

```
Import Workflow
```

Choose

```
Feedback Email Automation.json
```

---

### 5. Configure Gmail Credentials

Create OAuth Credentials from Google Cloud Console.

Fill in:

- Client ID
- Client Secret

Authenticate your Gmail account inside n8n.

---

### 6. Execute Workflow

Open the Form URL generated by the Form Trigger.

Submit:

- Name
- Email
- Feedback

You should receive an automatic thank-you email.
